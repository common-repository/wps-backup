Backup and migrate you WordPress website locally and via our API cloud service at WPScomplete.com. Simple, clean, hassle free: backup one or many websites easily.

== Description ==

Backup your websites database and restore with one click. Restore your backup with another click. Great for testing your website or keeping a fixed backup you can quickly restore.
Designed for simple, and reliable service that won't confuse you. This plugin is lightweight with all additional processes done by our server where possible.
Our free cloud sersvice backs up your site every week.  A manual backup to the cloud is also an option in the free subscription.  
The paid subscription allows multiple sites to be backed up more frequently and migration between two or more sites.
Requires <a href='https://wordpress.org/plugins/wps-complete/'>WPS Complete</a> to be installed for the API to our cloud service and debug tools.
Full guide to using the cloud backup option can be found <a href="https://wpscomplete.com/wps-backup-guide/" target="_blank" >HERE</a>

<br/>
Our full suite of plugins can be viewed at <a href='https://wpscomplete.com/plugins/'>WPS Complete Plugins</a>


== Installation ==

1. This plugin requires the WPS Complete plugin to be installed <a href='https://wpscomplete.com/plugins/'>WPS Complete Plugins</a>
1. Upload WPS Backup to wp-content/plugins
1. Click "Activate" in the WordPress plugins menu
1. Select the WPS Complete navigation, select the Backup submenu option
